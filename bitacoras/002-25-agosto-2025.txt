PROCESO TÉCNICO CLAVE - TOOLTIPS EN EJES:
------------------------------------------

PROBLEMA INICIAL: 
- Los tooltips aparecían en toda el área del plot usando IsPlotHovered()
- Se necesitaba restricción específica a los ejes del gráfico

INVESTIGACIÓN:
- Búsqueda manual en archivos fuente de ImPlot usando herramientas Grep y Read
- Identificación de la función IsAxisHovered(ImAxis axis) 
- Análisis de parámetros de ejes disponibles (ImAxis_X1, ImAxis_Y1)

IMPLEMENTACIÓN:
- Separación de lógica de detección por eje individual
- Uso de variables booleanas para claridad del código
- Condicionales específicas para cada caso (X solo, Y solo, ambos)
- Obtención de coordenadas del mouse con GetPlotMousePos()

CÓDIGO CLAVE:
```cpp
bool x_axis_hovered = ImPlot::IsAxisHovered(ImAxis_X1);
bool y_axis_hovered = ImPlot::IsAxisHovered(ImAxis_Y1);

if (x_axis_hovered || y_axis_hovered) {
    ImPlotPoint mouse_pos = ImPlot::GetPlotMousePos();
    ImGui::BeginTooltip();
    // Lógica específica por eje...
    ImGui::EndTooltip();
}
```
